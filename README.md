# Java学習用リポジトリ

Java（静的型付け言語）を習得するための学習記録です。

## 📚 学習ログ

### 1. Javaの基本 (section1_basics)
- 文字列の出力 (`System.out.println`)
- 数値の計算（整数同士の割り算は切り捨てられる点に注意）
- 文字列連結 (`+` 演算子)

### 2. 変数とデータ型 (section2_variables)
- **String**: 文字列型
- **int**: 整数型
- **double**: 小数型
- 変数の定義、更新、自己代入 (`+=`)
